What is the Document Object Model (DOM)?

The DOM is a structured model of a webpage that allows JavaScript to modify its content, style, and structure instantly.

How do you select an element from the DOM?

There are 5 ways we can do that.

  1. 1. document.getElementById()
  2. 2. document.getElementsByClassName()
  3. 3. document.getElementsByTagName()
  4. 4. document.querySelector()
  5. 5. document.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where we can add one event listener to the parent instead of adding event listeners to each child. It is a useful technique because it improves performance and leads to simpler code.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, we can use methods like getAttribute, setAttribute, and removeAttribute. For styles, we can change a style or retrieve it using getComputedStyle. For example, to change the background color of an element, we can use document.getElementById('idName').style.backgroundColor = 'blue'
